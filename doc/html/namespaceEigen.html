Detailed Description

We redefine all algebra and gemoetry types to hold double or single precision floats. The appended letter indicating this is 's', so that we have, e.g.,

  • VectorXf Vector of floats - defined by Eigen
  • VectorXd Vector of doubles - defined by Eigen
  • VectorXs Vector of either double of float, depending on the type scalar_t, defined by Wolf.
