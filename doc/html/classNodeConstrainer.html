template<class UpperType , class LowerType >
void NodeConstrainer< UpperType, LowerType >::computeError ( )
inlinevirtual

Calls computeError() recursively to down nodes, until reaching BOTTOM node, where computeError() is overloaded with the actual computation of its error.

  • For nodes TOP and MID, this method recursively calls computeError() of down nodes.
  • For nodes placed at the BOTTOM of the tree, this method needs to be overloaded (see e.g. CorrespondenceBase::computeError()).

The result of this call is placed at the member vector error_, which is an Eigen::Map to an external, contiguous, error_storage vector.

Reimplemented in CorrespondenceBase, and EpipolarConstraint.
